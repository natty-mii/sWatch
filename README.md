## :globe_with_meridians:アプリ名
sWatch(昔懐かし高橋名人のスウォッチ)

## :computer:概要
10間でどれだけボタンを押せるか？を測定するアプリ

## :speech_balloon:制作背景
Udemyでswiftコースを受講した際のアウトプットを行うために
本アプリを制作。<br>
お手本となるスォッチの画像もイラストレーターにて
自作。<br>
ボタンのカウントアップを学んだ際のアプトプットとして本アプリを  
作成しました。
